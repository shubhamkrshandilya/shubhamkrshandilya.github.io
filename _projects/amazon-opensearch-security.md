## Overview

As a **Software Development Engineer 2 at Amazon OpenSearch Service**, I specialize in the **security domain**, where I'm passionate about developing robust security features to enhance the protection of customer data and infrastructure. My work focuses on implementing critical security enhancements that serve millions of customers globally.

## Key Contributions

### Security Domain Leadership
- **TLS 1.3 & Perfect Forward Secrecy**: Led the implementation and rollout of TLS 1.3 protocol support with Perfect Forward Secrecy (PFS) cipher suites, significantly enhancing security posture for millions of OpenSearch clusters worldwide
- **Data Protection**: Developed robust security features to protect customer data both at rest and in transit
- **Infrastructure Security**: Enhanced security measures across OpenSearch Service infrastructure to defend against evolving cyber threats
- **Security Protocol Migration**: Designed and executed seamless migration strategies for existing customers to adopt enhanced security features without service disruption

### Performance Optimization
- Improved connection establishment performance with TLS 1.3's reduced handshake latency
- Optimized cluster management operations for large-scale deployments
- Enhanced monitoring and observability for security-related metrics

### Technical Leadership
- Published technical blog post on AWS Big Data Blog: ["Enhance security and performance with TLS 1.3 and Perfect Forward Secrecy on Amazon OpenSearch Service"](https://aws.amazon.com/blogs/big-data/enhance-security-and-performance-with-tls-1-3-and-perfect-forward-secrecy-on-amazon-opensearch-service/)
- Collaborated with cross-functional teams across security, infrastructure, and product management
- Mentored team members on security best practices and distributed systems design
